# 🎓 Student Performance Dashboard

A simple and interactive web application to analyze student performance using JavaScript array methods.

---

## 🚀 Features

- 📋 View all students  
- ✅ View passed students  
- ❌ View failed students  
- 📊 Calculate total marks  
- 📈 Calculate average marks  

---

## 💡 Concepts Used

This project helped me understand and implement:

- `map()` – for transforming data  
- `filter()` – for selecting specific data  
- `reduce()` – for calculating total values  
- `forEach()` – for iterating over arrays  
- DOM Manipulation  
- Event Handling  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 📸 Output

The dashboard displays:
- Student names  
- Filtered results (pass/fail)  
- Total and average marks

- Connecting UI with logic
